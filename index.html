Materi Yang Akan Dipelajari

Kamu akan dibimbing hingga menguasai ilmu UI/UX Research & Design dari dasar sampai penerapannya dalam pekerjaan

Dalam materi ini kamu akan mempelajari :
  • Sejarah Aplikasi
  • Mengenal Konsep Produk
  • Konsep Back-end, Front-ent, and REST API
  • Mengenal SDLC
  • SCRUM
  • MVP
  • Mengenal Tech Stack
  • Mengenal Bahasa Pemrograman
  • Memilih dan Memahami Tech Stack

Dalam materi ini kamu akan mempelajari :
  • Apa itu UX Design
  • UX itu bukan UI
  • UX bukan hanya tentang User
  • Kenapa harus memperhatikan UX
  • Design Handoff

Dalam materi ini kamu akan mempelajari :
  • Empathy dan pentingnya research dalam UX
  • User Interview (in-depth)
  • User Persona
  • Customer Journey
  • Flow Chart
  • Wireframe & Wireflow
  • Mockup (High Fidelity)
